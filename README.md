# Git Reminder – هرگز فراموش نکن push کنی روی گیت!

یه برنامه کوچیک و فوق‌العاده سبک که همیشه تو پس‌زمینه ویندوز اجرا میشه و **هر وقت Visual Studio یا VS Code رو ببندی**، یه پیام خوشگل میاره:

> **"پوش کردی رو گیت؟ اگه نکردی یادت نره"**

### ویژگی‌ها
- آیکن سبز خوشگل با حرف **G** تو تسکبار (قابل کشیدن و پین کردن به سمت چپ تسکبار ویندوز ۱۱)
- کاملاً مخفی و بدون پنجره
- فارسی ۱۰۰٪ درست نمایش داده میشه (با ایموجی)
- فقط ۸–۹ مگابایت (با UPX فشرده شده)
- با ویندوز استارت میشه (اوتو استارت)
- بدون نیاز به نصب – فقط یه فایل exe

---

### English

# Git Reminder – Never forget to push again!

A tiny, super-lightweight background app that runs on Windows and shows a friendly reminder **every time you close Visual Studio or VS Code**:

> **"Did you push to Git? Don't forget if you didn't!"**

### Features
- Beautiful green **G** icon in taskbar (can be dragged & pinned to the left side in Windows 11)
- Completely hidden – no window
- Full Persian/Unicode + emoji support
- Only **~8–9 MB** (compressed with UPX)
- Starts with Windows (auto-start)
- No installation needed – single .exe file

## نحوه فعال کردن اجرا با ویندوز (Auto Start)

### روش ۱ – ساده و سریع (پیشنهاد من)
1. کلیدهای `Win + R` رو بزنید
2. اینو تایپ کنید و Enter بزنید:

3. فایل `GitReminder.exe` رو بکشید و بندازید تو این پوشه (یا یه شورت‌کات ازش بسازید و بندازید)
4. تموم! از این به بعد هر بار ویندوز بالا بیاد، خودش اجرا میشه

### روش ۲ – حرفه‌ای (Task Scheduler)
اگه می‌خوای حتی قبل از لاگین هم اجرا بشه، از Task Scheduler استفاده کن (اختیاری)
